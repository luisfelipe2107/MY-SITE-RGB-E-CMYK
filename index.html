RGB e CMYK: O que são e como funcionam

Entenda como os sistemas de cores RGB e CMYK são usados em diferentes contextos, como em telas digitais e impressões.

O que são RGB e CMYK?

RGB (Red, Green, Blue) e CMYK (Cyan, Magenta, Yellow, Key) são sistemas usados para representar cores de maneiras diferentes. O sistema RGB é utilizado principalmente em telas digitais, enquanto o sistema CMYK é usado em impressão. Vamos explorar como cada um deles funciona.

RGB: O Sistema de Cores Aditivo

O modelo RGB é um sistema aditivo de cores, ou seja, ele começa com a ausência de cor (preto) e cria cores a partir da adição de luz nas cores primárias: Vermelho (R), Verde (G) e Azul (B). Ao adicionar intensidades diferentes de luz dessas cores, criamos todas as outras cores visíveis.

Como Funciona?

No modelo RGB, cada cor é representada por três valores, que podem variar de 0 a 255. Esses valores correspondem à intensidade de cada cor primária (R, G, B). O valor "0" significa a ausência de cor, e o valor "255" significa a máxima intensidade de luz.

Por exemplo:

  • rgb(255, 0, 0) - Vermelho
  • rgb(0, 255, 0) - Verde
  • rgb(0, 0, 255) - Azul
  • rgb(255, 255, 255) - Branco
  • rgb(0, 0, 0) - Preto

CMYK: O Sistema de Cores Subtrativo

O modelo CMYK é um sistema subtrativo de cores, ou seja, ele começa com uma folha branca e cria cores a partir da remoção de luz. As cores primárias no sistema CMYK são Ciano (C), Magenta (M), Amarelo (Y) e Preto (K), sendo o "K" utilizado para obter um preto mais intenso.

Como Funciona?

No modelo CMYK, cada cor é representada por uma porcentagem de cada tinta: Ciano, Magenta, Amarelo e Preto. O valor varia de 0% a 100%, sendo 0% a ausência de tinta e 100% a intensidade máxima.

Por exemplo:

  • cmyk(100%, 0%, 0%, 0%) - Ciano
  • cmyk(0%, 100%, 0%, 0%) - Magenta
  • cmyk(0%, 0%, 100%, 0%) - Amarelo
  • cmyk(0%, 0%, 0%, 100%) - Preto

Uma mistura de cores no sistema CMYK resulta na subtração de luz, o que significa que, ao adicionar mais cores, a luz vai diminuindo.

Comparação entre RGB e CMYK

Ambos os sistemas são usados para criar cores, mas em contextos diferentes:

  • RGB: Ideal para telas digitais como monitores, TVs e câmeras digitais. Funciona com luzes aditivas.
  • CMYK: Ideal para impressões em papel e outras superfícies. Funciona com pigmentos e é um sistema subtrativo.

Por exemplo, ao criar uma imagem digital, utilizamos RGB, enquanto ao imprimir essa imagem, utilizamos o sistema CMYK para obter as cores desejadas na impressão.
